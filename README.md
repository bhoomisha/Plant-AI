
## 🚀 Overview

PlantAI is a SaaS-style frontend platform that simulates AI-based plant detection, disease analysis, and personalized care recommendations.

It demonstrates real-world frontend engineering practices including scalable architecture, state separation, performance optimization, and persistent UX design.

---

## ✨ Core Features

### 🌱 AI Plant Scanner
- Upload plant images
- Simulated AI detection engine
- Confidence score generation (70–98%)
- Care + health recommendations

### 📊 Analytics Dashboard
- Real-time activity simulation
- Recharts-based graphs
- Plant health distribution insights
- Scan history tracking

### 🌿 Plant Intelligence System
- 53+ plant dataset
- Watering, sunlight, soil guidance
- Disease simulation engine
- Difficulty-based classification

### ❤️ Favorites System
- Add/remove plants
- Redux Toolkit state management
- Persistent localStorage storage

### 🔐 Authentication Flow
- Login / Signup UI
- Form validation using React Hook Form
- Protected route simulation
